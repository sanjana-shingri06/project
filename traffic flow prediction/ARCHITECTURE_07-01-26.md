## Data Flow Diagram

### 1. Real-Time Data Generation

```
Real-Time Data Generator Task (5s interval)
    в”Ӯ
    в”ңв”Җв–ә Iterate through all roads
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җв–ә Generate vehicle_count (30-400)
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җв–ә Calculate congestion_level
    в”Ӯ   в”Ӯ   в”ңв”Җ <50 vehicles вҶ’ 10%
    в”Ӯ   в”Ӯ   в”ңв”Җ 50-100 вҶ’ 30%
    в”Ӯ   в”Ӯ   в”ңв”Җ 100-200 вҶ’ 50%
    в”Ӯ   в”Ӯ   в”ңв”Җ 200-300 вҶ’ 70%
    в”Ӯ   в”Ӯ   в””в”Җ >300 вҶ’ 100%
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җв–ә Calculate speed
    в”Ӯ   в”Ӯ   в””в”Җ Speed = SpeedLimit - (Congestion% Г— 0.2)
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җв–ә Insert to traffic_updates table
    в”Ӯ   в”Ӯ
    в”Ӯ   в””в”Җв–ә Broadcast via WebSocket
    в”Ӯ       в””в”Җ emit('realtime_update', data)
    в”Ӯ           в””в”Җ Received by all connected clients
    в”Ӯ
    в””в”Җв–ә Wait 5 seconds, repeat
```

### 2. Analytics Processing

```
Analytics Engine (On-Demand)
    в”Ӯ
    в”ңв”Җв–ә Forecasting
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Fetch last 48h data
    в”Ӯ   в”ңв”Җ Group by road
    в”Ӯ   в”ңв”Җ Calculate mean
    в”Ӯ   в”ңв”Җ Calculate trend
    в”Ӯ   в””в”Җ Predict 6h ahead
    в”Ӯ
    в”ңв”Җв–ә Anomaly Detection
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Group last 24h by road
    в”Ӯ   в”ңв”Җ Calculate mean & std dev
    в”Ӯ   в”ңв”Җ Find values > 2Пғ from mean
    в”Ӯ   в””в”Җ Flag as anomaly
    в”Ӯ
    в”ңв”Җв–ә Health Scoring
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Score = 100 - (Congestion Г— 0.6) - ((50 - Speed) Г— 0.8)
    в”Ӯ   в”ңв”Җ Status = Excellent/Good/Fair/Poor
    в”Ӯ   в””в”Җ Return scores for all roads
    в”Ӯ
    в”ңв”Җв–ә Environmental Impact
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Total vehicles (24h sum)
    в”Ӯ   в”ңв”Җ Emission Index = Vehicles Г— Congestion / Speed
    в”Ӯ   в”ңв”Җ Fuel Efficiency = 100 - (Congestion Г— 0.8)
    в”Ӯ   в””в”Җ COвӮӮ = Vehicles Г— 0.2 Г— (100 - Efficiency) / 100
    в”Ӯ
    в”ңв”Җв–ә Peak Hours Analysis
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Group last 7 days by hour
    в”Ӯ   в”ңв”Җ Calculate avg congestion per hour
    в”Ӯ   в”ңв”Җ Find hour with max congestion
    в”Ӯ   в””в”Җ Return peak_hour for each road
    в”Ӯ
    в”ңв”Җв–ә Route Optimization
    в”Ӯ   в”Ӯ
    в”Ӯ   в”ңв”Җ Fetch current congestion all roads
    в”Ӯ   в”ңв”Җ Calculate estimated time = 10 / (Speed) Г— 60
    в”Ӯ   в”ңв”Җ Sort by congestion (ascending)
    в”Ӯ   в””в”Җ Return top 3 best routes
    в”Ӯ
    в””в”Җв–ә Recommendations
        в”Ӯ
        в”ңв”Җ IF Congestion > 70 THEN "Route Optimization"
        в”ңв”Җ IF Congestion 60-70 THEN "Monitor"
        в”ңв”Җ IF Anomaly DETECTED THEN "Investigation"
        в””в”Җ Return recommendations with priority
```
